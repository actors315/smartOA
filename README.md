项目引入了其他开源框架或类库，如下

```
引入thinkphp3.2  
```
cd $ROOT
git clone https://github.com/top-think/thinkphp.git
```
引入requirejs
```
cd $ROOT
git clone https://github.com/requirejs/requirejs.git
```
引入curl
```
cd $ROOT/SmartExtend/Weixin
