# 其他
项目引入了其他开源框架或类库并未添加到该项目源码中，需自行下载，如下  
引入Codeigniter3.x
```
cd $ROOT
git clone https://github.com/bcit-ci/CodeIgniter.git
```
引入thinkphp3.2  
```
cd $ROOT
git clone https://github.com/top-think/thinkphp.git
```
